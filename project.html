For this project we were tasked with creating a font hierarchy and design UI cards. I chose to use two fonts I am using right now: Poppins for headlines and Open Sans for paragraphs. I made this decision because I want To have a simple and easy to read website and sans serif fonts suit that role the best.

In addition to the choosing my fonts and setting their size, I also designed a set of UI cards. I used a very subtle drop shadow that grows when the card is hovered to create the illusion of movement.
